📝 Project Overview

Tudu App is a simple and user-friendly task management application that helps users organize their daily activities efficiently. Users can create an account, log in securely, manage their profile, and perform task operations such as adding, editing, deleting, and marking tasks as completed.

The application is designed to improve productivity by providing an easy way to track and manage personal tasks.

🚀 Features
🔐 User Authentication (Login & Signup)
👤 User Profile Management
➕ Add New Tasks
✏️ Edit Existing Tasks
❌ Delete Tasks
✅ Mark Tasks as Completed
📋 View All Tasks
🔎 Simple and Clean User Interface
📱 Responsive Design
